# Piano Sheet Music Hosting

Static assets for embedding rendered piano sheet music in Notion.

## Files

- `assets/c-major-scale.png` - Notion-friendly rendered image.
- `assets/c-major-scale.svg` - SVG render source.
- `index.html` - Simple GitHub Pages preview.

## Notion Embed URL Pattern

After publishing with GitHub Pages, use:

```text
https://<github-username>.github.io/<repo-name>/assets/c-major-scale.png
```

Use the PNG URL for Notion image blocks.
